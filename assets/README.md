# 图标资源说明

此目录存放小程序 TabBar 图标。

## 图标要求

- **尺寸**: 81px × 81px
- **格式**: PNG
- **大小**: 不超过 40KB

## 需要的图标

1. `home.png` - 首页图标（未选中）
2. `home-active.png` - 首页图标（选中）
3. `rewards.png` - 梦想屋图标（未选中）
4. `rewards-active.png` - 梦想屋图标（选中）
5. `stats.png` - 统计图标（未选中）
6. `stats-active.png` - 统计图标（选中）
7. `moments.png` - 美好图标（未选中）
8. `moments-active.png` - 美好图标（选中）
9. `profile.png` - 我的图标（未选中）
10. `profile-active.png` - 我的图标（选中）

## 临时方案

如果还没有准备图标，可以：
1. 使用 Emoji 作为临时图标（修改 app.config.ts）
2. 使用在线工具生成图标：https://iconscout.com/unicons
3. 使用 Figma 或 Sketch 设计图标

## 图标设计建议

- 使用简化的图标风格
- 选中状态使用亮色（#EC4899）
- 未选中状态使用灰色（#9CA3AF）
